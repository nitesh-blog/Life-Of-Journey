# Life-Of-Journey
This folder contains web pages created using MERN Stack. This folder contains a Life OF Journey. I have used:

1.Mongo DB
2.ExpressJD
3.ReactJS
4.NodeJs

for creating this web site as tour app. Its Fully Responsive.
